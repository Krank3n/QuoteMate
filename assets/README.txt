QuoteMate Assets

This folder should contain the following image files:

1. icon.png (1024x1024px)
   - App icon
   - Square PNG with rounded corners applied by iOS/Android

2. splash.png (1242x2436px minimum)
   - Splash screen image
   - Centered on dark background (#0F172A)

3. adaptive-icon.png (1024x1024px)
   - Android adaptive icon
   - Foreground layer only

4. favicon.png (48x48px or larger)
   - Web favicon

For now, the app will work without these.
You can use any image as placeholder or create proper branded assets later.

Recommended tools:
- Figma (free)
- Canva (free)
- Icon generator: https://icon.kitchen/

Theme colors:
- Primary (Eucalyptus Green): #009868
- Secondary (Golden Wattle): #cfa153
- Background (Dark): #0F172A
- Surface (Dark): #1E293B
